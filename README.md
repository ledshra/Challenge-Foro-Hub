# Challenge-Foro-Hub
Challenge foro hub:
## Se uso
Para la solucion del challenge se uso lo siguete:
- java version 17.
- Spring Boot version 3.4.1.
- intellij IDEA.
- y MySQL.

## Dependencias de Spring Boot
- Lombok
- Spring Web
- Spring Boot DevTols.
- SpringData JPA.
- Flyway Migration.
- MySQL Driver.
- Validation.
- Spring Security.

## Funcionalidades
    1 Registro de un nuevo tema.
    2 Mostrar todos los temas.
    3 Detallando un tema.
    4 Actulizar un tema.
    5 Eliminar un tema.
